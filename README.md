## 주제 : 갱년기 여성을 위한 건강 관리 앱
### 개발 기간
2023년 2월 25일 ~ 2023년 9월 12일
### 사용 기술
Spring Boot, AWS EC2, AWS RDS, AWS S3, FCM

### 주요 기능
- 로그인 및 회원가입: jwt와 spring security를 이용
- 약 알림: FCM(firebase cloud messaging)을 이용
- 약 복용 후 사진을 첨부하여 인증: AWS S3를 이용
- 채팅: 웹소켓을 이용
- 자유게시판
- 자가 진단 체크리스트
